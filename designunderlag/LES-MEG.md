# Designunderlag (historisk)

Disse notatene er den opprinnelige designbriefen for nettsidene, fra Nordic Storage-tiden.
De beskriver fortsatt gammelt navn og gammel fargepalett, og er **utdatert som fasit**.

Beholdt som referanse for produkt-/innholdslogikk (mål, konfigurator-algoritme, modelltabell osv.).
For gjeldende merkevare og farger, se `../../00-Merkevare/Visuell-profil.md`.
